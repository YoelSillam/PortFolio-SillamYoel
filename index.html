Explication : La balise <img> est utilisée pour insérer une image dans une page HTML. Dans ce cas, la source de l'image est définie par l'attribut src, qui pointe vers le fichier "WhatsApp Image 2023-06-15 at 12.27.01 (1).jpeg". Cette image sera affichée sur la page. Ensuite, nous avons la balise <map> qui est utilisée pour définir une carte d'image. La carte d'image permet de définir des zones spécifiques de l'image sur lesquelles l'utilisateur peut cliquer. La balise <map> doit avoir un attribut name qui spécifie le nom de la carte d'image. Dans ce cas, le nom de la carte d'image est "image-map". À l'intérieur de la balise <map>, nous avons la balise <area>. La balise <area> est utilisée pour définir une zone spécifique de l'image cliquable. Elle doit être placée à l'intérieur de la balise <map>. Les attributs importants de la balise <area> sont les suivants : target: Spécifie où ouvrir le lien après le clic. Dans ce cas, il est vide, ce qui signifie que le lien s'ouvrira dans la même fenêtre. alt: Texte alternatif affiché si l'image ne peut pas être chargée. title: Titre de la zone de l'image. href: L'URL de destination lorsque la zone est cliquée. Dans ce cas, il est vide, ce qui signifie qu'il n'y a pas de lien associé à cette zone. coords: Les coordonnées de la zone sur l'image. Les coordonnées spécifiques sont fournies sous forme de paires de nombres séparées par des virgules, représentant les points x et y. Ces coordonnées définissent les contours de la zone cliquable. shape: Spécifie la forme de la zone cliquable. Dans ce cas, la forme est "poly", ce qui signifie une zone polygonale. Les coordonnées dans l'attribut coords décrivent les points qui délimitent une zone polygonale sur l'image. Ces coordonnées sont utilisées pour définir la forme de la zone cliquable. Dans ce cas, la zone est un polygone avec plusieurs points définis.
